get_tensor_value(as_list=True)

Get value for onnx tensor.

Parameters:

as_list – whether return numpy ndarray in list.

Returns:

If as_list=True, return the array as a (possibly nested) list. Otherwise, return data of type np.ndarray.

If a tensor is a scalar having value 1,

when as_list=False, return np.array(1), type is <class ‘numpy.ndarray’> when as_list=True, return 1, type is <class ‘int’>.
